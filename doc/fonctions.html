Fonction Afficher

Syntaxe : AFFICHER [Variable Chaine de caractère]

Permet d'afficher à l'écran une phrase ou une variable.

Exemple :
AFFICHER Bonjour
-> Bonjour
CALCUL C=120
AFFICHER C
-> 120-

Fonction Lire

Syntaxe : LIRE [Variable]

Permet d'entrer une variable dans le programes.

Exemple :
LIRE A
-> A ?
AFFICHER A
-> 100

Fonction Calcul

Syntaxe : CALCUL [Variable]=[Variable|Nombres|Signe d'operation]

Permet d'effectuer un calcul.

Exemple :
CALCUL A=100
CALCUL C=120
AFFICHER A
-> 100
CALCUL C=A+C
AFFICHER C
-> 220

Fonction Set

Syntaxe : Set [Option]

Permet de modifier un parametre du programme.

Exemple :
SET DEG
SET RAD
